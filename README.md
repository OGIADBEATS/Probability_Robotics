# 概要
このプログラムは, Q学習を用いてロボットが1次元の数直線上を移動し, ゴール地点に到達するための最適な行動を学習するデモンストレーションである.

#機能説明
* ロボットの動作
  * 設定した開始位置から数直線上を移動し, ゴールに到達する機能.
  * 移動は「左移動 (L)」または「右移動 (R)」である.

* Q学習の実装
  * ロボットが行動と報酬を基にQ値を更新.
  * 学習率、割引率、探索率などのパラメータを調整可能.

* 出力
  * 学習後, ゴールまでの最適な行動がコンソールに表示される.
```
Optimal Policy:
R R R R G
```
